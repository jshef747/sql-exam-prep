# Starting point: fresh learner, exam-driven

No prior SQL knowledge has been demonstrated yet — treating this as ground zero. The mission is exam-output-driven (write correct SQL on demand), so lessons start from select-from-where and build toward the hard recurring exam patterns (division/"for all", outer-join-with-zeros, cross-group comparison).

The dominant practice schema is the **car-insurance** schema (person/owns/car/accident/garage); it appears in the most exams, so Lesson 1 uses it to make every example exam-relevant. See [[MISSION.md]] and `reference/glossary.html`.

Implications: until the user shows fluency with basic select/where/distinct, do not assume joins or subqueries. Re-assess ZPD after Lesson 1's self-check.
