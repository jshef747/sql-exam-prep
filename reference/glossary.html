איך קוראים את המסמך הזה

הטקסט ההסברי בעברית נכתב מימין לשמאל. כל קטע קוד SQL נכתב באנגלית בלבד ומיושר משמאל לימין — בדיוק כפי שתכתבו אותו בבחינה. שמות טבלאות ועמודות לעולם אינם מתורגמים לעברית. דוגמה לקוד:

select fname, lname
from person
where city = 'Haifa'
כלל הזהב: עברית = הסבר. אנגלית LTR = שפת המכונה. אם אתם רואים מילה כמו where או group by — היא תמיד באנגלית.

2 · מבנה שאילתה ומשמעות כל פסוק

שאילתת SQL בסיסית בנויה משלושה פסוקים, ושקולה לביטוי באלגברת היחסים (מצגת פרק 3, שקופית 3):

select  A1, A2, ... An      -- π  : which columns
from    r1, r2, ... rm      -- ×  : which tables
where   condition           -- σ  : which rows

  πA1..An ( σcondition ( r1 × r2 × ... × rm ) ) 

פסוקתפקידהערה
selectאילו עמודות יוחזרוselect * = כל העמודות. distinct מסיר כפילויות.
fromמאילו טבלאותכמה טבלאות מופרדות בפסיק = מכפלה קרטזית.
whereאילו שורות (תנאי סינון)השמטתו = כל השורות.
group byקיבוץ לקבוצותפונקציות הקבצה פועלות על כל קבוצה.
havingתנאי על קבוצותכמו where, אבל אחרי ההקבצה.
order byמיון התוצאהasc עולה (ברירת מחדל) / desc יורד.
סדר ביצוע לוגי: from → where → group by → having → select → order by.
